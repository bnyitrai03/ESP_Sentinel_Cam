QR Code Decoder

The QR Code Decoder component is used to decode QR codes from images. It is based on the quirc library.

The QR code format is the following:

WIFI_SSID|WIFI_PASSWORD|CONFIG_SERVER_URL

Where:

  • WIFI_SSID: The name of the Wi-Fi network.

  • WIFI_PASSWORD: The password of the Wi-Fi network.

  • CONFIG_SERVER_URL: The URL of the configuration server.

After decoding these will be saved in the NVS storage.

Header File

  • components/qr/include/qr_decoder.h

Classes

class QRDecoder

Handles the QR decoding.

Public Functions

QRDecoder(int width, int height)

Constructor to initialize the QR decoder.

The constructor initializes the quirc library and creates a quirc object.

Parameters:

  • width – Width of the camera frame.

  • height – Height of the camera frame.

bool decode_frame(const camera_fb_t *frame)

Function to decode the QR code from the camera frame.

The function extracts the camera frame buffer and decodes the QR code using the quirc library. The function returns true if the QR code is decoded successfully, otherwise false.

Parameters:

frame – Camera frame buffer.

Returns:

  • True if the QR code is decoded successfully

  • False if the QR code is not decoded
