CPU Temperature Sensor

The CpuTemp class provides an interface for interacting with the ESP32’s internal CPU temperature sensor. It implements the ISensor interface and supports initialization, reading, and retrieving the CPU temperature in degrees Celsius.

Usage Example

CpuTemp sensor;
if (sensor.init() == ESP_OK) {
    if (sensor.read() == ESP_OK) {
        float temperature = sensor.get_value();
        printf("CPU Temperature: %.2f°C\n", temperature);
    }
}

Header File

  • components/sensors/include/cpu_temp.h

Classes

class CpuTemp : public ISensor

CPU temperature sensor class.

Public Functions

virtual esp_err_t init() override

Initializes the CPU temperature sensor.

Returns:

ESP_OK if the sensor was initialized successfully, ESP_FAIL otherwise.

virtual esp_err_t read() override

Reads the CPU temperature.

Returns:

ESP_OK if the temperature was read successfully, ESP_FAIL otherwise.

virtual float get_value() const override

Returns the CPU temperature.

Returns:

The CPU temperature in degrees Celsius.
