bool wait_for_header_ack(const char *expected_timestamp)

Waits for an acknowledgment message with a specific timestamp, which will be sent upon receiving the header message.

Note

This function blocks until the acknowledgment is received or until the 5 second timeout

Parameters:

expected_timestamp – The expected timestamp for the acknowledgment

Returns:

  • true : if the acknowledgment with the expected timestamp is received

  • false : if the acknowledgment is not received within the timeout
