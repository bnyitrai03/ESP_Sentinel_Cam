HTTP Client

This component implements the HTTP client responsible for sending a GET request to the config server, to acquire the device’s static configuration.

The response static config JSON can be seen in the Config component.

Header File

  • components/communication/include/http_client.h

Classes

class HTTPClient

HTTP client class for sending GET request.

Public Static Functions

static esp_err_t get_config(const char *url, JsonDocument &response)

Sends a GET request to the specified URL and returns the static config as a JSON.

Parameters:

  • url – The URL to send the request to

  • response – JSON document to store the response

Returns:

  • ESP_OK on success

  • ESP_FAIL on failure
