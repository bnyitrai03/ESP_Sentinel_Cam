void restart()

Handles an error by restarting the device.

This function blinks the error LED pattern, then deinitializes the camera, MQTT, and Wi-Fi drivers. If the error_count is 15, the device will sleep for 10 minutes before restarting. Otherwise, the device will restart immediately.

void reset_device()

Resets the device.

This function erases the NVS and restarts the device.

void deinit_components()

Deinitializes all components.

This function deinitializes the camera, MQTT, and Wi-Fi drivers.
