Storage

The Storage component provides a simple key-value storage API for storing and retrieving data in NVS flash memory.

Note

The NVS is cleared whenever the device undergoes a reset.

Header File

  • components/storage/include/storage.h

static esp_err_t read_error_count(uint32_t *value)

Reads the error_count value from the NVS storage.

Note

If the error_count value does not exist, it will be created with a value of 1.

Parameters:
